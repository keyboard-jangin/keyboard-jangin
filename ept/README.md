# EPT (Equipment Performance Testing)

설비 성능 분석 프로그램

## 개요

EPT는 산업 설비의 성능을 측정, 분석, 보고하는 시스템입니다.

## 프로젝트 구조

```
ept/
├── src/
│   └── ept/                 # 메인 소스 코드
│       └── __init__.py
├── tests/                   # 테스트 코드
├── config/                  # 설정 파일
├── data/
│   ├── input/              # 입력 데이터
│   └── output/             # 출력 데이터
├── docs/                    # 문서
├── examples/                # 예제 코드
├── reports/                 # 분석 보고서
├── requirements.txt         # Python 의존성
└── README.md               # 이 파일
```

## 설치

```bash
pip install -r requirements.txt
```
